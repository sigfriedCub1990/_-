# _$
You Might Not Need jQuery
-------------------------
Don't get me wrong, [jQuery](https://jquery.org/) is awesome and I'm still learning how to unleash the full potential of jQuery, 
but let's face it, the [WebApi](https://developer.mozilla.org/kab/docs/Web/API) is awesome too and it's always a good
thing how to figure out how jQuery works under the hood.
This repository is inspired on the [Rob Rich Talk](https://www.youtube.com/watch?v=RdRMNT-98Oo&t=1442s) about this and in
this great [channel](youtube.com/user/techSithTube/videos) I've just found.
